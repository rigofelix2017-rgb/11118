## ğŸ†˜ Troubleshooting

### Button Not Showing?
- Make sure you've completed the intro sequence
- Ensure you've set up your profile
- Check if `gameStarted && userProfile` is true
- Try refreshing the page

### Keyboard Shortcuts Not Working?
- Make sure you're on desktop (not mobile)
- Check if another element has focus
- Try clicking the 3D canvas first
- Ensure CAPS LOCK is working (shortcuts work for both cases)

### System Won't Open?
- Check browser console for errors
- Make sure all 11 systems are in `components/systems/`
- Verify imports in `systems-hub.tsx`
- Try clearing browser cache

### Mobile Button Too Small?
- Button is 64x64px on mobile (should be comfortable)
- Check if device is detected as mobile (`isMobile` prop)
- Try landscape orientation for more space

### Animation Laggy?
- Disable animations in accessibility settings
- Close other browser tabs
- Check device performance
- Reduce graphics quality in settings
